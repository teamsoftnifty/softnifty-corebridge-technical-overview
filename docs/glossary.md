# Glossary

- **CBS**: Core Banking System  
- **ISO 20022**: Financial messaging standard for payments and reporting  
- **ISO 8583**: Legacy card/transaction messaging format  
- **ECO/ESG Telemetry**: Operational metrics used to estimate and report efficiency and sustainability indicators  
- **Design-Time**: Configuration stage where mappings/policies are prepared and validated  
- **Runtime**: Live transaction processing stage  
- **CoreRouter**: Routing decision component for selecting target rails and paths
