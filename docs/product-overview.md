# Product Overview (Public)

Softnifty AI‑Eco CoreBridge is an eco‑efficient middleware designed to connect legacy core banking systems to modern payment networks. It supports modern interoperability (including ISO 20022 readiness) and provides ESG/ECO telemetry to measure and improve operational efficiency.

## What CoreBridge does (high-level)
- integrates with existing core systems via connectors/adapters
- validates and normalises messages into a canonical form
- applies pre-approved mapping configurations
- enforces security and compliance controls (high-level)
- routes transactions to target rails
- captures telemetry for auditability and ESG reporting

## What this public portal does not include
This public portal intentionally excludes runtime source code and sensitive configurations. It is provided as a technical overview for transparency and collaboration discussions.
